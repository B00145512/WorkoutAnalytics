# Workout Analytics (in development)

## This software is a workout tool for novice people that go to a gym.

### *Features*
- GUI for workout selection
- Live Video feed of user with relevant landmarks connected and measured
- Accuracy score based on posture and repetition effort
- Written feedback for the user to improve in future sets
- Visualisations of their exercise with graphs based on previous uses, providing a comparison

Initially, it will prompt a user to select a workout and once it is selected, it will show a live video feed of the person and will signal the user into begining the workout.
Once the set is complete, *Workout Analyser* will give feedback based on how correct the posture is and offer feedback along with visualisation.
